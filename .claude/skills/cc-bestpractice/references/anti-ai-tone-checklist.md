## 中文文档 AI 味的典型特征（检查清单）

### 一、句式套路（最典型）

- [ ] **三段式滥用**：每段都"首先...其次...最后"、"一是...二是...三是"。真实写作不刻意凑三。
- [ ] **"不仅...而且"句式**："不仅是 X，更是 Y"。AI 最爱用，真人很少这么说话。
- [ ] **"值得注意的是"/"需要指出的是"**：hedging 词，删掉不影响信息。
- [ ] **"总而言之"/"综上所述"**：机械收尾。真人写作直接给结论。
- [ ] **"在某种程度"/"从某种意义上"**：模糊化，要么具体要么删。

### 二、结构套路

- [ ] **过度对称**：每个章节都是等长的 3-4 个子点。真实内容有长有短。
- [ ] **表格滥用**：能用句子说清的硬塞表格。
- [ ] **列表滥用**：把连贯逻辑拆成碎片化列表。
- [ ] **每节都"核心建议"收尾**：形式主义，不是每节都需要。

### 三、词汇套路

- [ ] **"赋能"/"助力"/"生态"/"闭环"/"抓手"**：互联网黑话，AI 尤爱。
- [ ] **"深度"/"全面"/"系统"**：形容词泛滥，不附具体证据。
- [ ] **"无缝"/"高效"/"强大"**：空话，无量化。
- [ ] **过度使用"的"**：长定语堆叠，"基于XXX的YYY的ZZZ"。

### 四、连接与过渡

- [ ] **机械过渡词**："接下来""然后""此外""另外"——真人写作过渡更自然或直接省略。
- [ ] **每段开头都是主题句**：太工整，像作文。
- [ ] **过度使用破折号**（——）：AI 偏爱，真人用得克制。

### 五、语气套路

- [ ] **过度中立**：没有立场，什么都"权衡"，不敢表态。
- [ ] **虚假共情**："我们知道你在想...""你可能会有疑问..."。
- [ ] **过度积极**：什么都"令人兴奋""前景广阔"。
- [ ] **缺乏个人痕迹**：没有第一人称经验、没有具体故事、没有判断。

## 去AI味的实操原则

1. **该断则断**：长短句混用，别都写长句。
2. **该具体就具体**：用"47 天里 46 天在用"代替"大量使用"。
3. **敢于表态**：有判断，不是永远"看情况"。
4. **少用结构**：能连贯说清的就别列表化。
5. **留个人痕迹**：第一人称、具体经历、独特观点。
6. **读出声**：读起来别扭的，多半是AI味。

## 核对命令（写作后自检）

写完一段，问自己：
- 删掉这句话，信息损失大吗？（不大就删）
- 这句话真人会这么说吗？
- 这段能不能换成一句直接的话？
- 我有具体例子/数字支撑吗？
